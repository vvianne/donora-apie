# Donora Final Project Documentation

## Overview

Donora is a digital blood coordination platform that connects hospitals, blood banks, registered donors, and transportation services to reduce emergency blood response time.

## Features

- **User Roles**: Donor, Hospital, Blood Bank, Transportation.
- **Core Functionalities**: Blood requests, inventory management, intelligent donor matching, transportation assignment, location tracking, and notification management.

## Technology Stack

- **Frontend**: React Native, React Navigation, Axios.
- **Backend**: Flask, SQLAlchemy, JWT, Marshmallow.
- **Database**: MySQL.
- **Deployment**: Docker, Docker Compose.
